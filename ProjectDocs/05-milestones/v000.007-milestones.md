### ``MS047 - exif-metadata-extraction``

**Type:**

- implementation

**Summary:**

- Align the EXIF module with the metadata extraction layer structure
- Move or integrate existing EXIF logic into the metadata_extractors structure
- Implement EXIF timestamp and timezone extraction according to defined field precedence and rules
- Ensure invalid EXIF data does not block fallback
- Ensure timezone is applied only to the corresponding EXIF timestamp field

**Covered Requirements:**

- R003 — EXIF Timestamp Extraction
- R004 — EXIF Timezone Extraction
- R002 — Metadata Extraction Layer Usage
- R016 — EXIF Module Structure Alignment

### ``MS048 - timestamp-resolution-pipeline``

**Type:**

- implementation

**Summary:**

- Implement pipeline behavior for all supported formats using a unified processing flow
- Ensure all supported formats pass through identical pipeline stages
- Prevent format-specific branching outside metadata extraction
- Implement timestamp source precedence and validity enforcement
- Ensure deterministic selection of the first valid timestamp source
- Enforce strict fallback behavior across all sources

**Covered Requirements:**

- R001 — Non-JPEG Format Processing Integration
- R005 — Timestamp Source Precedence
- R006 — Timestamp Validity Enforcement
